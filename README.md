## Git
* Git is a verison control
* Allows for various users to collaborate with eachother
* Can be runned any device you want
* Git itself is the application
## Github
* A web service where you can have an account
* Runs the Git software
* Github is a social network.
* **Repository**
  *   Another word for "Project"
  *   Allows for a user to view their history
  *   Can contain many files.
  *   Contains a file called **Read Me**
      * A **Read Me** is a special kind of file that goes in a respository that lets user know what the repository is about
      * Read Me's are made by default
      *   There are a lot of different file formats that **Git** and **Github** can keep track for you
  *   **Commit Files**
      *   Commit can be seen as another term for Change the file
      *   Creates a new file
      *   Everytime you commit you can browse the history and timeframe of when you made a change.
      *   You have your own hash for everytime you commit a file.
*   **Edit** button allows for file to be edited.
